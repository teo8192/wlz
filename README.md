# About

This is my attempt at implementing a Wayland compositor using [wlroots](https://gitlab.freedesktop.org/wlroots/wlroots/) that aims to have the same usage philosophy as [dwm](https://dwm.suckless.org) (using tags etc.).

The current goal is to re-implement [tinywl](https://gitlab.freedesktop.org/wlroots/wlroots/-/tree/master/tinywl?ref_type=heads) to learn about wayland and wlroots, and then going from there.
